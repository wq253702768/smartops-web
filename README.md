# smartops-web

This project contains a Vue 3 front-end application scaffold with an asset management module.
It demonstrates CRUD pages for customers, sites, remote access and host assets using
Vue Router, Pinia and @tanstack/vue-query. API calls are wrapped with an axios instance
for maintainable request management. Detail pages now fetch single records using dedicated
vue-query hooks.

Additional features include dynamic detail routing, filtering controls, password toggles and a simple login page supporting local and LDAP modes.
